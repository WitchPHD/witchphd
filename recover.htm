Note: This page is for my personal use. Effectively a backup to quickly re-setup my daily driver

This tarball should contain everything necessary to quickly set-up my system essentials in just about 15 minutes:

> run 01_installers.sh
> run 02_propagate.sh
> set [super + space] to start kitty
> set [alt + space] to open files
> set wallpaper
> set GNOME to dark theme
> set GNOME to purple accent color
> enable GNOME "night light"
