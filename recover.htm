Note: This page is for my personal use. Effectively a backup to quickly re-setup my daily driver
(after installing DE of choice - currently GNOME).

Arch repo installations (notes: uncomment [multilib] in etc/pacman.conf before, and enable ufw & fail2ban after):

sudo pacman -S ttf-hack-nerd noto-fonts-cjk tlp ufw fail2ban kitty starship neovim man-db gwenview firefox thunderbird libreoffice-fresh telegram vlc gimp steam transmission-qt python obs-studio v4l2loopback-dkms bluez

Install paru to install AUR packages (note: only the edvi-dkms-git version of displaylink works with my dock):

paru -S discord-canary displaylink nvchad-git

In neovim:

MasonInstall pylyzer

Finally, this tarball has my wallpaper, some config files, and some other stuff.
