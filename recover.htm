This page is just here in case something happens to my PC, so I can quickly re-install all my daily-driver software with a few commands.

First, an easy command to install all my applications from arch repo (note: uncomment [multilib] in pacman.conf for steam):

sudo pacman -S fastfetch power-profiles-daemon firefox thunderbird libreoffice-fresh telegram kitty vlc steam pika-backup gimp transmission-qt gwenview python obs-studio v4l2loopback-dkms ttf-hack-nerd noto-fonts-cjk

Secondly, let's set up paru for easy AUR installation:

sudo pacman -S --needed base-devel
git clone https://aur.archlinux.org/paru.git
cd paru
makepkg -si

Finally, the AUR package install (note: only the edvi-dkms-git version of displaylink works with my dock):

paru -S discord-canary displaylink
